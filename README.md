# AI Powered Library Management System

## Features

- Login
- Register User
- Add and Delete Books (Only for <b>Admins</b>)
- Views Books
- AI Recommend Books
- Issue and Return Books
- Search and Filter books
- Machine Learning Integration
- Database-driven storage
- Command Line Interface (<b>CLI</b>)

## Technologies used

- Python 
- MySql
- Machine Learning
- Git & GitHub

## Project Structure 

- AI_LMS_project.py - Main application
- cli.py - Command-line interface
- db.py - Database operations
- models.py - Data models
- requirements.txt - Dependencies 

## Instalation 

``` shell
git clone https://github.com/Sharan-2008/AI-Powered-Library-Management-System

cd AI-Powered-Library-Management-System

pip install -r requirements.txt

python AI_LMS_project.py
```
